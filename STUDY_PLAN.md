## 第一阶段：快速上手

- **重点**: 混个脸熟，不求精通，先能写出能跑的程序
- 状态：[完成]

## 第二阶段：基本概念

> 目标：理解 C 程序的构成要素——编译环境、词法规则、代码风格，为后续深入学习打下规范基础。

### 2.1 环境

- **内容**: 编译与链接过程（预处理→编译→汇编→链接）、GCC 工作流程、头文件与库文件搜索路径、集成开发环境
- **重点**: 理解从 `.c` 到 `.exe` 经历的四个步骤，知道每个步骤做了什么
- 状态：[完成]

### 2.2 词法规则

- **内容**: 字符集与编码、标识符命名规则、关键字、常量与字面量、运算符与分隔符、注释（`/* */` 和 `//`）
- **重点**: 合法的标识符是什么样；注释不能嵌套；三字母词（trigraph）的概念
- 状态：[完成]

### 2.3 程序风格

- **内容**: 缩进与空格、大括号风格（K&R / Allman / GNU）、命名规范、函数长度控制、注释原则、代码可读性
- **重点**: 风格可以不同，但必须统一；好的命名比注释更重要
- 状态：[完成]

## 第三阶段：数据

> 目标：透彻理解 C 语言的数据表示、声明与存储机制，这是嵌入式编程的基础。

### 3.1 基本数据类型

- **内容**: 整型家族、signed vs unsigned、浮点类型、各类型在不同平台的大小差异
- **重点**: 嵌入式平台上的类型大小可能和 PC 不同；`sizeof` 是你的尺子
- 状态：[完成]

### 3.2 基本声明

- **内容**: 变量声明语法、初始化 vs 赋值、声明指针/数组/函数、复杂声明的阅读方法
- **重点**: 学会"从内向外"阅读复杂声明
- 状态：[完成]

### 3.3 typedef

- **内容**: `typedef` 起别名、typedef vs `#define` 的区别
- **重点**: typedef 是为类型起新名字，不是创建新类型
- 状态：[完成]

### 3.4 常量

- **内容**: 字面常量、`const` 限定符、`#define` 符号常量、枚举常量
- **重点**: `const` vs `#define` 的区别；`const` 修饰指针的三种位置含义
- 状态：[完成]

### 3.5 作用域

- **内容**: 代码块作用域、文件作用域、函数作用域、原型作用域、变量遮蔽
- **重点**: 同名变量在不同作用域的行为；`-Wshadow` 能帮你发现遮蔽
- 状态：[完成]

### 3.6 链接属性

- **内容**: external / internal / none 三种链接属性、`extern` 和 `static` 的影响
- **重点**: 理解"声明"和"定义"的区别
- 状态：[完成]

### 3.7 存储类型

- **内容**: 自动存储（栈）、静态存储（静态区）、`auto`/`register`/`static`/`extern`
- **重点**: 栈变量只活在块内；静态变量只初始化一次
- 状态：[完成]

### 3.8 static 关键字

- **内容**: `static` 的三种用法——修饰函数内变量、文件内变量、函数
- **重点**: 同一个 `static`，三种语境三种含义
- 状态：[完成]

## 第四阶段：语句

> 目标：系统掌握 C 语言所有语句类型，理解它们的语义和执行流程。

### 4.1 空语句

- **内容**: 只有一个分号 `;` 的语句，占位和循环体为空的用法
- **重点**: 空语句的副作用——多写分号导致循环体变空
- 状态：[完成]

### 4.2 表达式语句

- **内容**: 表达式后加分号构成语句、赋值语句、函数调用语句
- **重点**: 表达式有值，语句没有
- 状态：[完成]

### 4.3 代码块

- **内容**: `{}` 包裹的复合语句、块内变量作用域、块嵌套
- **重点**: 代码块可出现在任何允许单个语句出现的地方
- 状态：[完成]

### 4.4 if 语句

- **内容**: `if`/`else if`/`else`、悬空 else、条件类型转换
- **重点**: else 与最近未匹配 if 结合；用 `{}` 避免歧义
- 状态：[完成]

### 4.5 while 语句

- **内容**: 先判断后执行、循环三要素、`break` 和 `continue`
- **重点**: 条件中做赋值（`while ((ch = getchar()) != EOF)`）是惯用法
- 状态：[完成]

### 4.6 for 语句

- **内容**: `for (init; cond; update)` 三段式、各段省略的语义
- **重点**: for 中声明的变量（C99）循环结束就消失
- 状态：[完成]

### 4.7 do 语句

- **内容**: 先执行后判断、至少执行一次、vs while 区别
- **重点**: do-while(0) 在宏定义中的妙用
- 状态：[完成]

### 4.8 switch 语句

- **内容**: `switch`/`case`/`default`/`break`、fall-through
- **重点**: 忘写 `break` 导致穿透是经典 bug
- 状态：[完成]

### 4.9 goto 语句

- **内容**: 无条件跳转、标签语法、goto 的正当用途
- **重点**: 跳出深层嵌套、错误处理统一清理点
- 状态：[完成]

## 第五阶段：操作符和表达式

> 目标：系统掌握 C 语言的操作符体系，理解表达式求值规则和类型转换机制。

### 5.1 操作符

- **内容**: 算术、关系、逻辑、位、赋值、条件（`? :`）、逗号、`sizeof`、类型转换操作符
- **重点**: 每个操作符的优先级与结合性；位操作与逻辑操作的区别
- 状态：[完成]

### 5.2 布尔值

- **内容**: C 中"真"和"假"的表示（非 0 即真）、`_Bool` 和 `<stdbool.h>`
- **重点**: `if (x = 5)` 这类误写永远为真；短路求值
- 状态：[完成]

### 5.3 左值和右值

- **内容**: 左值定义（可寻址）、右值定义（临时值）、哪些表达式是左值
- **重点**: 左值可出现在 `=` 左边；`++i` 是左值，`i++` 是右值
- 状态：[完成]

### 5.4 表达式求值

- **内容**: 求值顺序、副作用与序列点、短路求值（`&&` 和 `||`）、优先级总结
- **重点**: `a++ + a++` 是未定义行为；`&&` 左边为假则右边不执行
- 状态：[完成]

## 第六阶段：指针

> 目标：彻底掌握 C 语言指针，从内存模型到指针运算，这是 C 语言最核心的能力。

### 6.1 内存和地址

- **内容**: 内存模型、字节地址、地址与内容、指针就是存地址的变量
- **重点**: 画内存图是理解指针最有效的方法
- 状态：[ ]

### 6.2 值和类型

- **内容**: 变量的值和类型、指针变量的值和类型、类型决定解读方式
- **重点**: 同一个地址，类型不同解读不同
- 状态：[ ]

### 6.3 指针变量的内容

- **内容**: 指针变量里存的是一个地址（整数值）、`%p` 格式打印地址
- **重点**: 指针变量本身也在内存中有地址
- 状态：[ ]

### 6.4 间接访问操作符

- **内容**: `*` 解引用、`&` 取地址、`*&a` 和 `&*p` 的含义
- **重点**: `*` 用在声明和用在表达式是不同的含义
- 状态：[ ]

### 6.5 未初始化和非法的指针

- **内容**: 野指针、未初始化指针的解引用后果、`NULL` 的保护作用
- **重点**: 永远不要解引用未初始化的指针
- 状态：[ ]

### 6.6 NULL 指针

- **内容**: `NULL` 的定义、空指针检查、解引用 NULL 的后果（段错误）
- **重点**: 函数返回指针时用 `NULL` 表示失败
- 状态：[ ]

### 6.7 指针间接访问和左值

- **内容**: `*p` 是左值（可赋值）、指针间接访问作为左值的使用场景
- **重点**: `*p = 10` 修改的是 p 指向的那个变量
- 状态：[ ]

### 6.8 指针间接访问和变量

- **内容**: 通过指针间接修改变量、指针与变量的关系
- **重点**: 指针是访问变量的"另一条路"
- 状态：[ ]

### 6.9 指针常量

- **内容**: 数组名、字符串字面量、`const` 修饰指针的几种形式
- **重点**: `int *const p` vs `const int *p` vs `const int *const p`
- 状态：[ ]

### 6.10 指针的指针

- **内容**: 二级指针、三级指针、`int **pp` 的含义和用法
- **重点**: 每多一级 `*` 就多一次间接寻址
- 状态：[ ]

### 6.11 指针的表达式

- **内容**: 指针参与的表达式、`*p++`、`*++p`、`++*p` 等组合
- **重点**: 优先级和结合性决定表达式含义，不确定时加括号
- 状态：[ ]

### 6.12 指针的运算

- **内容**: 指针加减整数、指针相减、指针比较、`p + 1` 加的是 `sizeof(*p)`
- **重点**: 指针运算自动按所指向类型的大小缩放
- 状态：[ ]
