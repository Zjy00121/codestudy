# 第五章：操作符和表达式 — 知识点总结

## 5.1 操作符

### 算术操作符

- 两个整数相除执行**整除**（截断小数）：`5 / 2 = 2`
- 只要有一个浮点数就算浮点除法：`5 / 2.0 = 2.5`

### 位操作符

| 操作符 | 名称 | 规则 | 用途 |
|--------|------|------|------|
| `&` | 按位与 | 全 1 才 1 | 清零特定位、检测某位 |
| `\|` | 按位或 | 有 1 就 1 | 把某位置 1 |
| `^` | 按位异或 | 不同才 1 | 翻转特定位、交换值 |
| `~` | 按位取反 | 1 变 0，0 变 1 | 配合 `&` 清零 |
| `<<` | 左移 | 低位补 0 | 每移一位 = × 2 |
| `>>` | 右移 | unsigned 补 0；signed 补符号位（GCC） | 每移一位 = ÷ 2 |

### 左移和右移的关键区别

- **左移**：不管有符号还是无符号，低位一律补 0
- **右移**：`unsigned` → 逻辑右移（补 0）；`signed` → 实现定义（GCC 用算数右移，补符号位）
- 建议：位移操作统一用 `unsigned`，行为确定

### 位移量规则

位移量必须满足 `0 <= n < 位宽`：
```c
a << 32;   // ❌ 32 位 int 上 UB
a << -5;   // ❌ UB
```

### 位操作 vs 逻辑操作

| | 位操作 `&` `\|` `~` | 逻辑操作 `&&` `\|\|` `!` |
|------|---------------------|--------------------------|
| 操作对象 | 每一位 | 整体是非 0 |
| 返回值 | 位运算后的数值 | 只返回 0 或 1 |
| 短路求值 | 无 | `&&` 左边假→跳过右边；`\|\|` 左边真→跳过右边 |

```c
3 & 2  → 2          // 逐位与
3 && 2 → 1          // 都非 0，真
if (p && p->data)    // ✅ 短路保护
if (p &  p->data)    // ❌ 不短路，p 为 NULL 也崩
```

### 赋值操作符 `=`

- **右结合**：`x = y = z` → `x = (y = z)`，x 和 y 都被赋值
- **返回值**：赋值表达式的值 = 赋进去的值，因此 `if (x = 5)` 能编译且永远为真

### 单目操作符

| 操作符 | 名称 | 说明 |
|--------|------|------|
| `!` | 逻辑非 | 真变假，假变真 |
| `~` | 按位取反 | 每一位翻转 |
| `++` / `--` | 自增/自减 | 前缀先改后用，后缀先用后改 |
| `-` / `+` | 负号/正号 | 取反/不改变 |
| `&` / `*` | 取地址/解引用 | 单目含义不同于双目的按位与/乘 |
| `sizeof` | 大小 | 返回字节数，操作数可以不加括号 |
| `(type)` | 强制转换 | 转类型 |

> `++a = 10` 在 C 中非法（`++a` 是右值），在 C++ 中合法（`++a` 是左值）。

### 条件操作符 `? :`

C 唯一的**三目操作符**，需要返回值时替代简单 if-else：
```c
int max = (a > b) ? a : b;
```

### 逗号操作符 `,`

从左到右依次执行，返回**最右边的值**：
```c
x = (1, 2, 3);    // x = 3

// while 中的常见用法：先读再判断
while (printf("> "), fgets(line, 128, stdin) != NULL)

// for 中并行操作
for (i = 0, j = 10; i < j; i++, j--)
```

## 5.2 布尔值

- C 语言规则：**非 0 即真，0 即假**
- C99 起用 `#include <stdbool.h>` → `bool` / `true` / `false`
- 老编译器可自造：`#define BOOL int` 或 `typedef enum { FALSE, TRUE } bool_t;`

---

## 5.3 左值和右值

| | 左值 | 右值 |
|------|------|------|
| 定义 | 有持久内存位置，可取地址 | 临时值，无固定地址 |
| 测试 | `&(expr)` 编译通过 | `&(expr)` 编译报错 |
| 例子 | 变量、数组元素、`*p` | `10`、`a+b`、`a++`、`++a` |

```c
int a = 5;       // a 是左值，5 是右值
(a + b) = 5;     // ❌ 左值必须是可修改的
```

> 数组名是左值（可取地址 `&arr`）但不能放 `=` 左边。

## 5.4 表达式求值

### 优先级与结合性

- 高 → 低：单目 > 乘除 > 加减 > 移位 > 关系 > 相等 > 位操作 > 逻辑 > 条件 > 赋值 > 逗号
- 重点记忆：**位操作 `&` `|` `^` 在 `==` 之后**！`if (flag & 0x01 == 0)` 等于 `if (flag & (0x01 == 0))`，必须加括号

### 短路求值

`&&` 左边假 → 跳过右边；`||` 左边真 → 跳过右边。

### 序列点与副作用

**序列点**（`;`、`&&`、`||`、`?:`、`,`、函数调用前）之前的所有副作用会结算。

### 未定义行为（UB）

相邻两个序列点之间**同一个变量被修改两次** = UB：
```c
a++ + a++      // ❌ UB
i = i++;       // ❌ UB
arr[i++] = i;  // ❌ UB
```

## printf 格式

| 格式 | 类型 | 用途 |
|------|------|------|
| `%d` / `%i` | `int` | 有符号十进制 |
| `%u` | `unsigned int` | 无符号十进制 |
| `%x` / `%X` | `unsigned` | 十六进制 |

| `%c` | `char` | 字符 |

## 进制前缀

| 进制 | 前缀 | 示例 | 十进制值 |
|------|------|------|----------|
| 十进制 | 无 | `42` | 42 |
| 八进制 | `0` | `052` | 42 |
| 十六进制 | `0x` | `0x2A` | 42 |

> ⚠️ `0` 开头是八进制！`int n = 052` 不是 52。

## 经典函数实现

### reverse_bits — 位翻转（不依赖机器字长）

```c
unsigned int reverse_bits(unsigned int value) {
    unsigned int answer = 0;
    unsigned int i;
    for (i = 1; i != 0; i <<= 1) {  /* i 溢出归零自动适配位宽 */
        answer <<= 1;
        if (value & 1) answer |= 1;
        value >>= 1;
    }
    return answer;
}
```

### 位数组操作（set / clear / test / assign）

```c
void set_bit(char bit_array[], unsigned bit_number) {
    bit_array[bit_number / 8] |= (1u << (bit_number % 8));
}
void clear_bit(char bit_array[], unsigned bit_number) {
    bit_array[bit_number / 8] &= ~(1u << (bit_number % 8));
}
int test_bit(char bit_array[], unsigned bit_number) {
    return (bit_array[bit_number / 8] & (1u << (bit_number % 8))) != 0;
}
```

- `bit_number / 8` → 落在第几个字节
- `bit_number % 8` → 字节内第几位（0~7）

### store_bit_field — 位域存储

```c
width = starting_bit - ending_bit + 1;
mask = (width 个 1) << ending_bit;
original_value &= ~mask;          // 清空目标区域
value_to_store = (value_to_store << ending_bit) & mask;  // 对齐并截位
return original_value | value_to_store;
```

- 先构建掩码（先左移腾位再 `|=1`）
- 清空 → 对齐 → 截位 → 合并

### scanf 返回值陷阱

```c
while (scanf("%d", &n) == 1)   // ✅ EOF 时退出
while (scanf("%d", &n))        // ❌ EOF=-1 为真，死循环
```
